End-to-end Power BI dashboard built on 180,000+ real-world supply chain transactions, showcasing Power Query data cleaning, DAX measures, dynamic slicers, bookmarks, and geospatial mapping across two report pages. Covers sales performance ($36.78M total), profitability ($3.97M profit), delivery logistics, and customer geography, all cross-filtered and drill-through enabled. Key finding: LATAM and Europe post near-identical sales volumes, yet LATAM carries a disproportionately higher late delivery burden - a supply chain inefficiency the dashboard makes impossible to miss.
